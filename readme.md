## project name 
Spotify Popularity Prediction
## description
This project predicts whether a Spotify song will be popular based on its other features.
## Table of Contents

1- Imported the necessary libraries.
2- Analyzed missing data and removed unnecessary columns.
3- In the EDA (Exploratory Data Analysis) phase, created pastel-toned, infographic-style visualizations.
4- During feature engineering, created new variables such as sex, embarked, and family_size.
5- Split the dataset into training and testing sets using train_test_split.
6- Scaled the data using StandardScaler.
7- Built a LinearRegression model.
8- Trained the model and displayed its accuracy score.
9- Generated a confusion matrix visualization.
